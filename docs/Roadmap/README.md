# StudentOS Strategic Roadmap

Our roadmap flows through six strategic execution steps:

1. **StudentOS Foundation** (Current: Monorepo workspaces, types, Express & SQLite, dark UI).
2. **Planner** (Classic timetable scheduling, task lists, and goals tracking).
3. **AI Planner** (Priority scoring, spaced repetition scheduler, calendar availability optimizer).
4. **AI Agent** (LangGraph autonomies, Google integrations, email notifications).
5. **Personal OS** (Consolidated notebooks, files indexer, developers dashboards).
6. **Eventually SaaS** (Auth/user scopes, multi-tenant databases, Stripe integrations, Cloud Deploys).
